<!-- AJUI_Animate_info ( ) -> version -->


## Description

Show the version info of the component.

```4d
  AJUI_Animate_info ( ) -> version
```

| Parameter | Type | In/Out | Description |
| --------- | ---- | ------ | ----------- |
| version | text | out | version info of the component |

## Example

```4d
  C_String($version)
  
  $version:= AJUI_Animate_info () //AJUI_Animate (v18) : 1.0.0-Build69 
```
