# formResetObjectsCoordinates

## Parameters

 *  none

## Return value

none

## Description

This method moves and resizes all the form objects using the coordinates stored in the object variable **Form.AJUI_Animate_formObjects**. See formGetObjectsCoordinates.

## Example

```
 formResetObjectsCoordinates ( )
```
